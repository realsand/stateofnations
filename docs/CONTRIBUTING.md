# Contributing Guidelines

- Place new client scripts in `src/client/`
- Place new server scripts in `src/server/`
- Place shared code in `src/shared/`
- Use `modules/`, `components/`, and `util/` folders as appropriate
- Add assets to `assets/`
- Add tests to `tests/`
- Update documentation in `docs/`

Follow the folder structure for clarity and maintainability.
